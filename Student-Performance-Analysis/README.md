# 📊 Student Performance Analysis

## 🎯 Objective
Analyze student performance data using Python and statistics to identify trends, relationships, and performance factors.

---

## 🛠️ Skills
Python, Pandas, NumPy, Matplotlib, Seaborn, EDA, Statistics

---

## 📈 Analysis Performed
- Data Cleaning
- Statistical Analysis
- Distribution Analysis
- Correlation Analysis
- Outlier Detection using Z-score
- Probability Analysis
- Data Visualization

---

## 🔥 Key Highlights
- Performed statistical and correlation analysis
- Detected outliers using Z-score and boxplots
- Visualized score distributions and trends
- Analyzed impact of lunch type on performance
- Identified strong positive correlation between reading and writing scores

---

## 📊 Visualizations
- Histogram
- Boxplot
- Heatmap

---

## 🧠 Conclusion
This project helped identify performance patterns, score relationships, and the impact of socio-economic factors on student academic outcomes using statistical analysis and visualization.
