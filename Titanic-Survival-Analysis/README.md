# 🛳 Titanic Survival Analysis

## 📌 Objective
The goal of this project is to analyze the Titanic dataset and identify the key factors that influenced passenger survival.

---

## 🔍 Dataset
- Dataset Source: [Kaggle - Titanic: Machine Learning from Disaster](https://www.kaggle.com/c/titanic/data)
- Main file: `train.csv`

---

## 🧰 Tools & Libraries
- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib
- Jupyter Notebook

---

## 📊 Key Insights
- Women and children had higher chances of survival.
- Passengers in **1st class** survived at a much higher rate than those in **3rd class**.
- Older passengers generally had lower survival chances.
